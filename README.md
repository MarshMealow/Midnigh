# Midnight Motors — Car Marketplace Web App

## About
Midnight Motors is a car marketplace website that lets users browse, search and manage car listings.  The original project was built with **HTML**, **CSS**, **JavaScript**, **PHP** and **MySQL**.  Sellers can register and log in, add or remove their own vehicles, and see feedback.  Buyers can view listings, search for cars and leave reviews.

## Features
* Seller registration and login.
* Create, update and delete car listings with images and details.
* Search page with filters for make and model.
* Feedback and rating functionality.
* Responsive UI built with HTML and CSS.
* Sample database schema included in `midnight_motors.sql`.

## Backend
The full project uses PHP and MySQL for server‑side logic and data storage.  Database connection details are configured in `db.php`.  To run the project locally:

1. Import the `midnight_motors.sql` file into a MySQL database.
2. Update the credentials in `db.php` to match your database.
3. Serve the files through a local web server such as XAMPP or MAMP.
4. Open `index.php` in your browser via `localhost`.

## Static Demo
GitHub Pages does not support PHP, so this repository includes only the source code.  For a live demo you can host the static front‑end files separately or deploy the project to a PHP‑enabled hosting platform.
